# e-plantShopping


### Paradise Nursery - Shopping Application

Paradise Nursery is an online shopping application that allows users to browse various houseplants, add them to a shopping cart, and proceed to checkout. The application features dynamic cart functionality, allowing users to adjust quantities, view total costs, and manage items before checkout.

## Features

- **Landing Page**: Displays an introduction to Paradise Nursery with a background image and a Get Started button that links to the product listing page.
- **Product Listing Page**: Users can browse plants organized into categories, view plant details (name, image, price), and add items to the shopping cart.
- **Shopping Cart Page**: Displays the items added to the cart, allows users to adjust quantities, remove items, and shows the total number of items and total cost.
- **Responsive Design**: The application is mobile-friendly and adapts to different screen sizes.

## Pages

1. **Landing Page**:
    - Background image
    - Company name and description
    - "Get Started" button linking to the product page

2. **Product Listing Page**:
    - Display at least six plants, categorized into different types (e.g., Air Purifying, Medicinal, Aromatic).
    - Each plant includes a thumbnail image, name, price, and an "Add to Cart" button.

3. **Shopping Cart Page**:
    - Shows all plants in the cart with details like name, thumbnail, price, and quantity.
    - Allows users to increase/decrease quantity, remove plants, and displays the total cost.

## Installation

To get started with the project, clone the repository to your local machine:

```bash
git clone https://github.com/AnisulMahmud/e-plantShopping.git
cd e-plantShopping
npm install
npm run preview
```
The app is available in https://anisulmahmud.github.io/e-plantShopping/
